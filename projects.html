Exploratory Data Analysis of Stack Overflow Developer Survey 2020

I have done a lot of really interesting analysis in this project which I highly recommend taking a look at. This was a self-driven project. I found the Stack Overflow Developer survey to be interesting and I was personally curious about the responses of the participants like their favorite social media platforms, the distribution of countries or the programming languages they have worked with. I mainly used Python’s Pandas library to work with this data. I love this library very much; it offers a lot of flexibility in researching and manipulating data. Paired with Matplotlib or Seaborn, this library can be a powerful tool to create very intuitive and informative visualizations.

Design, Creation and Query of a Goodreads Database of Over 16000 Books

This SQL project was for one of my courses at school. I did everything from scratch. I obtained a dataset of Goodreads books, their authors, ratings, etc on Kaggle. Then I used Python’s Pandas to clean the dataset, remove unnecessary data, strip all the spaces and new lines, etc. Based on the final dataset that I wanted to use, I designed an Entity Relationship Diagram. I wrote a Python script to import the CSV file into a SQL database, and then I wrote queries to find the answers to my research questions. I had a lot of fun working on this project. All the files are uploaded onto my GitHub which include the dataset, the python script, python code for cleaning the dataset, the MySQL dump file and SQL queries.
